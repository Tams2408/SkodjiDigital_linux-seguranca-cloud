##Este repositório demonstra na prática as aulas de Linux e Segurança na Cloud da formação com a Skodji Digital.
#Skodji Digital é uma plataforma que foi desenvolvida para apoiar a implementação dos percursos de formação digital do programa, com o objetivo de reforçar as competências digitais dos participantes em Cabo Verde e promover o acesso a oportunidades no mercado digital global.
