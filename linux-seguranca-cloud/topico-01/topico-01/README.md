## Comandos executados

* pwd → mostra o diretório atual
* whoami → mostra o utilizador atual
* hostname → mostra o nome da máquina
* uname -a → mostra informações do kernel
* lsb_release -a → mostra a versão do Linux
* free -h → mostra memória RAM disponível

## Diferença entre VM, VPS e infraestrutura em nuvem

### VM

Máquina virtual executada localmente no computador.

### VPS

Servidor virtual privado executado remotamente num datacenter.

### Infraestrutura em nuvem

Recursos virtualizados distribuídos pela internet com escalabilidade e gestão centralizada.
