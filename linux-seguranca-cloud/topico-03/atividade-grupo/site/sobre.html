Sobre o Projeto

Este projeto tem como objetivo demonstrar a criação, publicação e validação de um serviço web em ambiente Linux.

O serviço foi publicado utilizando o servidor web Nginx, permitindo o acesso através de um navegador web.

Foram aplicados conhecimentos sobre:

  • Gestão de ficheiros Linux
  • Serviços Web
  • Publicação de conteúdos
  • Validação de serviços
  • Documentação técnica
